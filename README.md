### Online Appendices for "Workshop Report: Quantitative Bias Analysis for Differential Misclassification of Exposure (Recall Bias) in Practice"

This repository contains the online appendices accompanying the manuscript:

Workshop Report: Quantitative Bias Analysis for Differential Misclassification of Exposure (Recall Bias) in Practice

submitted to the Journal of Statistics and Data Science Education.

The materials provided here support transparency, reproducibility, and allow readers to reproduce the analyses presented in the manuscript using R. The repository includes both code-based appendices and one text-based appendix (Appendix F), as referenced in the manuscript.


### Repository Contents

- Appendix D
Setting parameters of Beta priors for misclassification parameters based on prior elicitation questions

- Appendix E
Probabilistic sensitivity analysis for differential exposure misclassification
(R code)

- Appendix F
Bayesian adjustment for differential exposure misclassification
(Text-based appendix)

### Package Information: episensr

- Title: Basic Sensitivity Analysis of Epidemiological Results
- Description: Provides tools for basic sensitivity analysis of observed relative risks, allowing adjustment for unmeasured confounding and misclassification of exposure and/or outcome. The package implements quantitative bias analysis methods described in: Lash TL, Fox MP, Fink AK. Applying Quantitative Bias Analysis to Epidemiologic Data. Springer, 2021.
- Author: Denis Haine [aut, cre] ORCID: https://orcid.org/0000-0002-6691-7335
